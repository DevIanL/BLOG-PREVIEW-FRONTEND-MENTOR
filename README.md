# 📰 Blog Preview Card - Frontend Mentor

![Status](https://img.shields.io/badge/Status-Concluído-success)

## 💻 Sobre o Projeto
Desenvolvimento de um card de preview para blog, baseado em um desafio do [Frontend Mentor](https://www.frontendmentor.io/). Focado em estruturação semântica e estilização de componentes de interface.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3

## 🚀 Como Executar
1. Clone o repositório na sua máquina.
2. Abra a pasta no editor de sua preferência.
3. Utilize a extensão **Live Server** para visualizar no navegador.
